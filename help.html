Vault Project

Vault is my take on the game launcher customization scene. It's a hobby project living off the free github storage space.
I love the steamgriddb.com and inspired a lot from them. Still uploading the low-resolution versions and a lot more to there.

I'm currently a sophomore student at a Computer Science (Eng.) Faculty in Turkey and trying to improve my skills with projects like this. You can check my other projects on my website.
